# TIL (Today I Learned)

개발/보안 직무 준비를 위해 학습한 내용을 짧게 정리하는 저장소입니다.  
목표는 "많이 기록"이 아니라 "다시 써먹을 수 있는 기록"입니다.

## 기록 원칙
- 코드/개념은 `왜 이렇게 했는지`까지 함께 적기
- 하루 단위 기록보다 `주간 요약`으로 재사용성 높이기
- 면접에서 설명 가능한 수준으로 정리하기

## 디렉토리 구조
```text
TIL
├── logs/2026/                  # 주간 학습 로그
├── security/                   # 보안 개념/점검 체크리스트
├── backend/                    # API/서버 설계 메모
├── algorithm/                  # Python 알고리즘 패턴 정리
├── ssafy/                      # SSAFY 프로젝트 회고/학습 포인트
└── templates/                  # 재사용 템플릿
```
